# KDM6AD 논문 페이지 색인

이 폴더는 Zotero 컬렉션 `KDM6+`에서 KDM6/KDM6AD와 직접 연결되는 42편을 논문별로 분리한 학술 계보 위키다. 각 페이지는 단순 초록 요약이 아니라 학술적 가치, 계보적 위치, 수학적 구조, KDM6AD의 JVP/VJP 또는 자료동화 연결, 정당화 가능한 주장과 불가능한 주장을 구분한다.

## 범위

- 전체 논문 페이지: 42편
- PDF 원문 확보: 35편
- 원문 보강 필요: 7편
- 원천 컬렉션: Zotero `KDM6+` / `ZABGLNPX`
- 작성 원칙: 논문을 “KDM6AD 원고용 인용 카드”가 아니라 KDM6/KDM6AD 과학 계보의 독립 노드로 다룬다.

## 페이지 읽는 법

- KDM/WDM 핵심 문헌은 forward parity와 state-vector 의미를 확인하는 데 먼저 읽는다.
- 미세물리 설계공간 문헌은 KDM6AD가 채택하지 않은 대안 closure까지 포함해 학술적 한계를 정한다.
- 자동미분 문헌은 discrete derivative, AD toolchain, parameter inference, differentiable programming 계보를 제공한다.
- 자료동화 문헌은 JVP/VJP가 observation-space loss와 만나는 지점을 설명한다.
- 원문 보강 필요 문헌은 보수적으로 계보 연결만 사용하고, 세부 수치 주장은 원문 확보 후 갱신한다.
